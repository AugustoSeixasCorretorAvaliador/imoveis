# Augusto Seixas Corretor - Site Imoveis

Repositorio com os arquivos estaticos do site `augustoseixascorretor.com.br`.

## Estrutura
- `index.html`: pagina inicial.
- `imovel/`: paginas de imoveis.
- `imovel-lp/`: landing pages de imoveis.
- `imagens/`: imagens e identidade visual.

## Publicacao no GitHub
1. Faça as alteracoes nos arquivos.
2. Rode:
   ```bash
   git add -A
   git commit -m "Sua mensagem"
   git push
   ```

## Backup rapido
Para gerar um ponto de restauracao:
```bash
git add -A
git commit -m "Backup YYYY-MM-DD"
git push
```
